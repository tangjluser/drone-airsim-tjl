# 2.使用

test1文件问仿真环境的启动

> cd test1
>
> 点击test1.sln
>
> 在右侧将game栏目下的test1设置为项目启动项
>
> 再将上方导航栏修改成 DebugGame  和 Win64 
>
> 最后点击本地Windows调试器，即可运行仿真环境
>
> 在环境中点击运行即可生成无人机

myproject为项目代码的主体

> cd myproject
>
> 运行dijkstra.py 即可规划出全局路径
>
> 运行base.py 就可运行
